# 🎮 Tetris Game

¡Bienvenido al juego de Tetris! Este es un clásico juego de rompecabezas donde las piezas de diferentes formas caen desde la parte superior del tablero. El objetivo es encajar las piezas para completar líneas horizontales, que luego se eliminan y te dan puntos. ¡Diviértete jugando! 🕹️

## 📦 Contenido del Proyecto

Este proyecto incluye los siguientes archivos:

- `index.html` - La estructura básica del juego.
- `styles.css` - Estilos visuales para el juego y sus elementos.
- `script.js` - La lógica del juego, incluyendo el movimiento de las piezas y la puntuación.

## 🎮 Cómo Jugar

1. **Iniciar el Juego**: Simplemente abre el archivo `index.html` en tu navegador.
2. **Controles**:
   - **⬅️ Izquierda**: Mueve la pieza a la izquierda.
   - **🔄 Girar**: Gira la pieza en sentido horario.
   - **➡️ Derecha**: Mueve la pieza a la derecha.
   - **⬇️ Abajo**: Mueve la pieza hacia abajo más rápido.

3. **Objetivo**: Completar líneas horizontales para eliminarlas y obtener puntos. ¡Intenta conseguir la mayor puntuación posible! 🎯

## 🌟 Características

- **Sistema de Puntuación**: Obtén puntos al eliminar líneas.
- **Dificultad Aumentando**: La velocidad de las piezas aumenta a medida que avanzas en el juego.
- **Responsive**: Diseñado para jugar en dispositivos móviles y de escritorio.

## 🚀 Instalación

1. Clona el repositorio en tu máquina local:
   ```bash
   git clone <URL_DEL_REPOSITORIO>
   ```
2. Abre el archivo `index.html` en tu navegador.

## 🎨 Personalización

Puedes personalizar el estilo del juego editando el archivo `styles.css`. Cambia colores, fuentes y tamaños para que se adapte a tu gusto. ¡Deja volar tu creatividad! ✨
